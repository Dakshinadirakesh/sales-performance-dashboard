# 📊 Sales Performance Dashboard

## Overview
This project presents an interactive Sales Performance Dashboard built using Power BI to analyze sales data and provide actionable business insights. The dashboard helps track key performance indicators (KPIs), identify sales trends, and support data-driven decision-making.

## Objectives
- Analyze overall sales performance.
- Track revenue, profit, and order trends.
- Identify top-performing products and regions.
- Compare sales across different categories and time periods.
- Enable interactive filtering for detailed analysis.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

## Dashboard Features
- Total Sales, Profit, and Orders KPIs
- Sales Trend Analysis
- Region-wise Sales Performance
- Category & Sub-category Analysis
- Top Products by Sales
- Interactive Slicers and Filters

## Key Insights
- Identified top-performing products and categories.
- Analyzed regional sales performance.
- Tracked monthly sales trends.
- Highlighted opportunities for improving business performance.
